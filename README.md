# AI Productivity Analyzer - Prompt Engineering Dashboard

##  Project Overview
An end-to-end analytics project measuring the real productivity impact of AI assistance on developer and analyst workflows. Built entirely from scratch using Excel and Power BI.

##  Problem Statement
Companies are investing heavily in AI tools but lack data to prove ROI. This project tracks and measures whether AI actually saves time, improves output quality, and which prompt styles work best.

##  Tools Used
- Microsoft Excel — data modeling, KPI formulas, pivot reports
- Power BI Desktop — interactive dashboard, DAX measures, slicers

##  KPIs Measured
-  Time Saved % - how much faster tasks are with AI
-  Quality Delta - does AI improve or hurt output quality?
-  Prompt Effectiveness Score - which prompt style wins?
-  AI ROI Flag - did AI deliver measurable value?

##  Data Model
4 connected Excel tables:
- Tasks - role, task type, complexity, team
- Productivity - time with/without AI, errors, iterations
- Prompts - prompt style, AI model, quality score
- Quality - accuracy score, rework needed, reviewer rating

## Key Findings
- Code Generation saves the most time - 64% faster with AI
- Role-Based prompts score highest across all AI models
- 3 out of 5 tasks (60%) showed positive AI ROI
- Average time saved across all tasks - 33%
